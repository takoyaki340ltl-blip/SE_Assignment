## 安裝與環境設定 (Setup)

### 安裝必要套件
```bash
pip install matplotlib
```

## 執行說明

### 1. 輸入紀錄 (Member A: 1122961 林冬騋)
執行 `input.py` 開始輸入資料：

```bash
python input.py
```
* **輸入格式**：`日期 金額 類別 [備註]`
* **範例**：`today 150 lunch 雞腿飯`
* **備註說明**：日期、金額、類別為必填項目，備註 (note) 為選填。
* **資料儲存**：所有輸入的資料將會自動儲存於 `data/expenses.csv`。
* **結束輸入**：輸入q即可結束。

### 2. 輸出圓餅圖 (Member B: 1122923 林科穎)
執行 `output.py` 開始輸出圓餅圖：

```bash
python output.py
```
* **輸出視窗**：依照各類別加總的金額匯至圓餅圖，並在下方顯示最近五筆紀錄。
* **清單呈現**：終端機會印出完整的消費紀錄清單。
